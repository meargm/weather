# 🌦️ Weather App

A simple weather app that shows current weather by city using OpenWeatherMap API.

## 🚀 Features
- Search by city
- Shows temperature, humidity, and condition
- Responsive design

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- OpenWeatherMap API

## 📦 Setup
1. Clone the repo
2. Replace `"YOUR_API_KEY"` in `script.js`
3. Open `index.html` in browser
